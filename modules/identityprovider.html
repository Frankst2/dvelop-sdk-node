getAuthSession

    • Defined in identityprovider/src/get-auth-session/get-auth-session.ts:22

    Returns an AuthSession based on an API-Key. The AuthSessionId should be kept secret and never be publicly available

    example
    const authSession: AuthSession = await getAuthSession("https://dharma-initiative.d-velop.cloud", "<API_KEY>");

console.log('still valid:', authSession.expire.getTime() > new Date().getTime()); //still valid: true
