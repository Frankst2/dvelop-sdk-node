getAuthSession

  • getAuthSession(systemBaseUri: string, apiKey: string): Promise<AuthSession>
    • Defined in packages/identityprovider/src/authentication/get-auth-session/get-auth-session.ts:26

    Returns an AuthSession based on an API-Key. API-Keys can be generated by d.velop cloud tenant administrators. The AuthSessionId should be kept secret and never be publicly available.

    throws

    UnauthorizedError indicates an invalid authSessionId or no authSessionId was sent.

    example
    const authSession: AuthSession = await getAuthSession("https://monster-ag.d-velop.cloud", "<API_KEY>");
console.log(authSession.id); //a valid authSessionId
console.log('still valid:', authSession.expire.getTime() > new Date().getTime()); //still valid: true

    Parameters

    • systemBaseUri: string

      SystemBaseUri for the tenant

    • apiKey: string

      API-Key provided by the d.velop cloud

    Returns Promise<AuthSession>

validateAppSessionSignature

    • Defined in packages/identityprovider/src/authentication/validate-app-session-signature/validate-app-session-signature.ts:26

    Validate the sign value which is provided when an appSession is sent. For further information on this process refer to the documentation.

    throws

    InvalidAppSessionSignatureError indicates that the sign value is not valid

    example
    acceptPostRequest(req: Request, res: Respone) {
  try {
    validateAppSessionSignature("acme-app", "requestIdUsedOnAppSessionRequest", req.body.authSessionId, req.body.expire, req.body.sign); //pass or error
  } catch(e) {
    res.send("Forbidden");
  }
}

      The expire value provided to your callback
