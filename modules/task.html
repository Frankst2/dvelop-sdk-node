completeTask

  • completeTask(systemBaseUri: string, authSessionId: string, task: string | Task): Promise<void>
    • Defined in packages/task/src/complete-task/complete-task.ts:31

    Mark a Task as completed.

    throws

    NoTaskLocationError indicates that no location was given.

    throws

    UnauthenticatedError indicates that the authSessionId was invalid or expired.

    throws

    UnauthorizedError indicates that the user associated with the authSessionId does miss permissions.

    throws

    TaskNotFoundError indicates that for the given location no task was found.

    throws

    TaskAlreadyCompletedError indicates that a task is already marked as completed.

    example
    
await completeTask("https://umbrella-corp.d-velop.cloud", "AUTH_SESSION_ID", "/some/task/location");

// or

const task: Task = {
  location: "/some/task/location",
  ...
}
await completeTask("https://umbrella-corp.d-velop.cloud", "AUTH_SESSION_ID", task);

      SystemBaseUri for the tenant

createTask

  • createTask(systemBaseUri: string, authSessionId: string, task: Task): Promise<Task>
    • Defined in packages/task/src/create-task/create-task.ts:30

    Create a Task.

    This method will automatically generate a random correlation key if the task does not contain one.

    throws

    InvalidTaskError indicates that the given task was not accepted because it is invalid. You can check the error.validation-property.

    throws

    UnauthenticatedError indicates that the authSessionId was invalid or expired.

    throws

    UnauthorizedError indicates that the user associated with the authSessionId does miss permissions.

    example
    
const task: Task = {
  subject: "Cover up lab accident",
  assignees: ["USER_ID_1", "USER_ID_2"],
  correlationKey: "everythingIsFine", // can be randomly generated
}
task = await createTask("https://umbrella-corp.d-velop.cloud", "AUTH_SESSION_ID", task);

    Parameters

    • systemBaseUri: string

      SystemBaseUri for the tenant

    • authSessionId: string
    • task: Task

      Task to be created

    Returns Promise<Task>

    Created Task
