validateRequestSignature

  • validateRequestSignature(appSecret: string, systemBaseUri: string, tenantId: string, requestSignature: string): boolean
    • Defined in app-router/src/validate-request-signature/validate-request-signature.ts:18

    Validates a request-signature against your appSecret. The requestSignature should be validated for every request when recieving calls from the d.velop cloud. Refer to the d.velop basics tenant header section for more information.

    example
    const valid: boolean = validateRequestSignature("ptuQ0b0BskmLLxXsjjhH9Su8ozTvZl6Z/5/HlaORoRg=", "https://header.example.com", "a12be5", "Zjcf28p5aQ6amtbs6s9b9cPyBPdziwUslR2DZqaGUTQ=");
console.log(valid); //true

    Parameters

    • appSecret: string

      The secret generated for your app in the cloud-center

    • systemBaseUri: string

      SystemBaseUri for the tenant (found in 'x-dv-baseuri'-header)

    • tenantId: string

      ID for the tenant (found in 'x-dv-tenant-id'-header)

    • requestSignature: string

      ID for the request (found in 'x-dv-sig-1'-header)

    Returns boolean
