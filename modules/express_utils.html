redirectToLoginPage

  • redirectToLoginPage(req: Request, res: Response): void
    • Defined in packages/express-utils/src/functions/redirect-to-login-page/redirect-to-login-page.ts:31

    Redirect a request to the login page provided by the Identityprovider-App.

    import { redirectToLoginPage, UnauthorizedError } from "@dvelop-sdk/express-utils";

app.use((err, req, res, next) => {
  if (err instanceof UnauthorizedError && req.accepts("text/html")) {
    redirectToLoginPage(req: Request, res: Response);
  } else {
    next();
  }
});
